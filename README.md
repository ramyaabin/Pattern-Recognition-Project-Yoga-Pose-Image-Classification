## Project Overview
This project classifies yoga poses using Deep Learning techniques including Convolutional Neural Networks (CNNs) and Transfer Learning.

## Dataset
Yoga Pose Image Classification Dataset:
https://www.kaggle.com/datasets/shrutisaxena/yoga-pose-image-classification-dataset

## Technologies

- TensorFlow/Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Dataset exploration
- Image preprocessing
- Data augmentation
- CNN model training
- Transfer Learning
- Performance evaluation
- Confusion matrix visualization

## Results
Model successfully classifies multiple yoga poses with high accuracy.
